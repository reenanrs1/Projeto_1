Oque é a Tag Head?

É uma parte fundamental de qualquer documento HTML.

Ela é usada para conter metadados sobre o documento e outras informações importantes que não são exibidas diretamente na página,

mas que influenciam como o navegador e outros sistemas interpretam e processam o conteúdo.

Tudo o que está dentro da tag headserve para definir configurações, adicionar funcionalidades, ou melhorar o desempenho da página.

Oque é a Tag Title?

É a descrição da página.

É a primeira coisa que o usuário vê quando ele abre a página, e é usado pelo navegador para definir o título da página na barra de título.

Oque é a Tag Body?

É a parte principal da página.

É onde você coloca o conteúdo principal da página, como texto, imag ens, vídeos, formulários, etc.

Quais são as tags de edição de texto?

Tag b e strong

Tag b

A tag b é usada para tornar o texto em negrito. Ela não tem importância semântica, ou seja, apenas muda a aparência visual do texto.

Tag strong

A tag strong também deixa o texto em negrito, mas ela carrega um significado semântico. Isso significa que o texto envolvido com strong é considerado importante.

Tag i

A tag i é usada para colocar o texto em itálico, mas sem implicar qualquer ênfase ou importância semântica.

Tag Em

A tag em também coloca o texto em itálico, mas com ênfase semântica. Isso significa que o texto dentro de em é considerado relevante ou enfatizado.

Tag u

A tag u sublinha o texto, mas não tem um valor semântico específico.

Tag ins

Indica que o texto foi inserido ou adicionado recentemente. Ele geralmente é sublinhado, mas também tem um significado semântico, sugerindo que o conteúdo foi incluído como uma correção ou inserção

Tag s

A tag A tag s (Strikethrough) risca texto, indicando que ele foi desconsiderado ou que não é mais relevante, sem carregar significado semântico.

Tag del

A tag del indica que o texto foi excluído ou removido, carregando um significado semântico de exclusão

Tag mark

A tag mark é usada para destacar o texto, geralmente com uma cor de fundo amarela, como se estivesse marcado com um marcador de texto. Ela tem um valor semântico, indicando que o conteúdo é importante ou relevante no contexto atual.

Tag small

A tag small é usada para exibir texto em um tamanho menor do que o texto padrão, geralmente para informações secundárias ou observações.

Tag sub

A tag sub é usada para criar texto subscrito, geralmente utilizado em fórmulas químicas ou matemáticas.

Tag sup

A tag sup é usada para criar texto sobrescrito, comum em expoentes ou anotações em cima de caracteres.

Tag Q

A tag q é usada para inserir uma citação curta em linha, que geralmente é exibida entre aspas.

Tag blackquote

A tag blockquote é usada para inserir uma

citação
longa, que é geralmente exibida com indentação, separada do corpo principal do texto.

Tag code

A tag code é usada para exibir pequenos trechos de código dentro do texto. Ele preserva a formatação de texto monoespaçada.
